# rke2-vagrant

Create a kubernetes cluster using vagrant/kvm/rke2 in minutes.

## Prerequisite
```
$ vagrant plugin install vagrant-disksize
```

## Usage
### Stop
```
$ sudo vagrant destroy
```
## Usage
### Start
```
$ git clone https://github.com/derekbit/rke2-vagrant.git
$ cd rke2-vagrant
$ sudo ./start.sh
```

### Stop
```
$ sudo ./stop.sh
```
